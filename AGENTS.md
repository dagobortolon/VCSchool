# Project Instructions

## Image Persistence
- The `/public/images` directory contains essential assets uploaded by the user.
- **NEVER** delete or overwrite files in `/public/images` unless explicitly requested.
- Ensure that this directory is always tracked in version control (GitHub).
- The naming convention for images is strictly defined in `/public/images/README.md`.

## Language Settings
- The application defaults to English (`en`) on load.
- Language switching is handled via UI interaction.
