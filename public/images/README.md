# Imagens do Site

Coloque as imagens nesta pasta com os nomes exatos abaixo para que apareçam automaticamente no site. **Não altere os nomes após o upload.**

## Identidade Visual
- `logo_vini_cavalcanti_3D.svg` (Logo da Escola)
- `favicon_vini_cavalcanti.png` (Ícone do navegador)

## Seções
- `hero_image.webp` (Imagem da Seção Hero - 16:11)
- `about_vini_photo.webp` (Foto da Seção Sobre - 490x760)
- `mentorship_image.webp` (Imagem da Seção Mentoria - 21:9)

## Cursos (Exemplos)
- `course_retopologia_in_topoGun3.webp` (Miniatura do curso de Retopologia)
- `course-Zbrush_For_Stylized_Characters.webp` (Miniatura do curso de ZBrush)

---
*Nota: Caso o curso não tenha uma imagem específica com nome personalizado, ele tentará carregar um padrão baseado no título ou o arquivo configurado em `src/constants.ts`.*
